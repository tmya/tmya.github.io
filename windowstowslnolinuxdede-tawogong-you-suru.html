WindowsとWSLのLinuxでデータを共有する

LinuxからWindowsのデータにアクセスしたりWindowsからLinuxのデータにアクセスする方法を忘れそうになるのでメモです。

Windows -> Linux

WindowsからLinuxのデータ、つまりファイルを開くにはエクスプローラーのアドレス欄に\\wsl$を入力します。するとインストールしたディストリビューションごとに共有フォルダが表示されるので、開くと各データにアクセスできます。

Linux -> Windows

Windowsのドライブが最初から/mnt以下にマウントされているので、それぞれドライブレターのディレクトリを開くとデータが入っています。
