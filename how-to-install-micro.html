micro(テキストエディタ)のインストール - MIRACLE LINUX 8.4

viやEmacsは難しい？ではテキストエディタのmicroをインストールしよう。

インストール

インストールはとっても簡単。2つのコマンドで完了する。

$ curl https://getmic.ro | bash
$ sudo mv micro /usr/bin

使い方

起動方法

単体で起動すると空のエディタが立ち上がります。

$ micro

後ろにファイル名を指定するとそのファイルを開くことができます。

$ micro hoge.txt

操作方法

ほとんどWindowsのメモ帳と同じ。カーソルキーで動かしたり文字を打てば入力できます。

保存等

Ctrl + s で保存。Ctrl + qで終了。microのみで起動した後、Ctrl + sを押すとファイル名を聞かれるので入力してEnterキーで保存。Ctrl + qを押すと変更を保存するか聞かれるので、yかnかキャンセルならEscを押す。

後はmicro エディタ 使い方で検索すると各種ショートカットが調べられると思います。
