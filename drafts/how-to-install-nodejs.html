Node.jsのインストール - MIRACLE LINUX 8.4

Node.jsをインストールしたい。

手順

  1. Node.jsのインストール？

Node.jsインストール？

私自身はNode.jsを触ったことがほとんどありません。人気らしいのでてっきり最初からMLにインストールされているのかなと思いコマンドを打ったところインストールされていませんでした。その代わりにインストールを促すメッセージが表示されました。便利ですね。

$ node
bash: node: コマンドが見つかりませんでした...
コマンド node' を提供するためにパッケージ 'nodejs' をインストールしますか? [N/y] y

yと入力しEnterキーを押すだけでインストールが完了しました。

$ node -v
v10.24.0

現時点ではv10.24.0がインストールされるようです。
