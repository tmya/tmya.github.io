DaVinci Resolve 17のインストール - MIRACLE LINUX 8.4

DaVinci Resolveは動画編集ソフトです。

前提

  • EPELリポジトリが使えること

  • 使えない場合はMIRACLE LINUX 8.4 でいろんなリポジトリを使ってみよう｜BLOG｜サイバートラストの下の方に記載の方法で使えるようにしましょう。コマンド一発です。

インストール

$ unzip ./DaVinci_Resolve_17.4.5_Linux.zip
$ ./DaVinci_Resolve_17.4.5_Linux.run 
$ sudo dnf install ocl-icd
