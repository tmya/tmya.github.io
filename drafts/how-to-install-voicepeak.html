VOICEPEAK 商用可能 6ナレーターセットのインストール - MIRACLE LINUX 8.4

VOICEPEAK 商用可能 6ナレーターセット｜製品情報｜AHS(AH-Software)はテキストから音声を出力できるソフトウェアです。

前提

  • EPELリポジトリが使えること

  • 使えない場合はMIRACLE LINUX 8.4 でいろんなリポジトリを使ってみよう｜BLOG｜サイバートラストの下の方に記載の方法で使えるようにしましょう。コマンド一発です。

インストール

unzipコマンドで解答すると文字化けしてしまうので、MacOSでは有名なunarで解凍します。コマンドラインツールだけならWindowsやLinuxでも使用できます。上記の通りEPELが使えるのであれば下記コマンドでインストール可能です。

$ sudo dnf install unar

解凍したフォルダへ移動します。

$ cd VOICEPEAK\ 商用可能\ 6ナレーターセット\ ダウンロード版

解凍したフォルダのはじめにお読みください.txtの内容を確認すると手順は次の通りだそうです。

  • ダウンローダの Linux/voicepeak-downloader-linux64 を実行する。

  • 画面にしたがってダウンロード開始とアクティベーションコードの入力を行う。

  • ダウンロードしたインストーラを実行する。

  • インストールを行って完了。

Linux/voicepeak-downloader-linux64を実行する

実行権限がついてないのでそのままでは実行できません。
