Homebrewのインストール - MIRACLE LINUX 8.4

  1. Homebrewのインストール

Homebrewのインストール

数か月前まではLinuxbrewというものがありました。今はHomebrewに統合されています。

公式サイトに手順が載っています。とても簡単です。下記コマンドを実行するとsudoが使えるか確認のためユーザのパスワードが聞かれます。

パスワードを入力するとチェックが行われます。

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

下記のメッセージが表示され、Enterキーを押すとダウンロードとインストールが始まります。

Press RETURN to continue or any other key to abort:

==> Installation successful!が表示されたら完了です。

echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> /home/tmya/.bash_profile
$ eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"
$ sudo yum groupinstall 'Development Tools'
