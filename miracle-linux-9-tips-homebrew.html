HomebrewをMIRACLE LINUX 9で使う方法

HomebrewをMIRACLE LINUX 9で使う方法

別の記事で用意したWSL上のMIRACLE LINUX 9にHomebrewを導入します。もちろん通常通りインストールされたMIRACLE LINUX 9の環境でも同じように導入できます。これで大抵の開発環境は簡単に揃えることができるはずです。

基本的に公式サイトの手順通りです。

注意 : 以下のコマンド中の /home/user/.bash_profile のuserは2行とも自分のユーザ名に置き換えてください。

$ sudo dnf install git
$ bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
$ echo '# Set PATH, MANPATH, etc., for Homebrew.' >> /home/user/.bash_profile
$ echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> /home/user/.bash_profile
$ eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"
$ sudo dnf groupinstall 'Development Tools'
$ brew install gcc
